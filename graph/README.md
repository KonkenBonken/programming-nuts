`@Programmeringsnöten` #9

Skriv ett program där en funktion `f(x: nummer): nummer` är givet. Graften `y = f(x)` är simpel kurva med en maxpunkt.
Därefter ska ditt program hitta x-värdet för denna maxpunkt.

Exempel 1:

```ts
f(x): -x²
Svar: 0
```

Exempel 2:

```ts
f(x): -28x² + 42x
Svar: 0,75
```
